# Demo Video Script

## Introduction

Hi, my project focuses on classifying the quality of discourse in NBA fans' reactions on Twitter/X during Game 3 of the 2018 NBA Finals between the Cleveland Cavaliers and Golden State Warriors.

I fine-tuned DistilBERT to distinguish between high-quality discourse, which contains reasoning or evidence, and low-quality discourse, which consists mainly of emotional reactions or unsupported claims.

## Correct Prediction Example

One example that the model classified correctly was:

> "Golden State's switching defense forced Cleveland into isolation possessions."

The model predicted High Quality with 91% confidence.

This prediction makes sense because the tweet contains actual basketball reasoning. It explains how Golden State's defensive strategy affected Cleveland's offense and provides a causal relationship rather than just expressing emotion.

This is exactly the kind of analytical discourse I intended the model to recognize.

---

## Incorrect Prediction Example

One incorrect prediction involved the tweet:

> "JR Smith just cost LeBron the game."

The true label was High Quality, but the model predicted Low Quality.

This error likely occurred because the tweet is short and emotionally phrased. During training, many short and emotional tweets were labeled as low quality, so the model learned to associate those patterns with the Low Quality class.

As a result, it relied more on surface features than on the intended distinction between emotional reactions and evidence-based arguments.

---

## Evaluation Report

For evaluation, I compared my fine-tuned DistilBERT model with a zero-shot Llama-3.3-70B baseline.

| Model                   | Accuracy |
| ----------------------- | -------- |
| Zero-shot Llama-3.3-70B | ~0.77    |
| Fine-tuned DistilBERT   | ~0.47    |

At first glance, the zero-shot model appears much better.

However, the labels in the dataset were originally generated by Llama-3.3-70B itself. This means the zero-shot baseline was effectively being tested against its own answers, which inflates its accuracy.

Meanwhile, DistilBERT had to learn from only 151 automatically generated labels, many of which contained noise.

Because of this, I don't interpret the results as evidence that zero-shot prompting is inherently better than fine-tuning. Instead, the project showed me that data quality and independent ground truth are more important than simply choosing a larger model.

---

## Closing

Overall, this project taught me that evaluation metrics can be misleading when labels are noisy or generated by the same model being evaluated. In future work, I would create a larger dataset with human-reviewed annotations to obtain a more reliable comparison between zero-shot and fine-tuned models.
